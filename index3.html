About This Project

Final Year Dissertation — Cloud Computing · Intelligent Scheduling · Fault Tolerance

Research gap addressed

Most prior work treats fault tolerance and intelligent scheduling as separate concerns. This project unifies them — GA optimises globally, then the fault-tolerance layer handles failures with greedy min-load recovery, without re-running the expensive search.

Two-stage modular design: GA optimisation + fault recovery layer. No performance penalty in fault-free conditions beyond small FT readiness overhead.
GA fitness function
f(chromosome) = α · makespan + β · imbalance
α = 0.7 (makespan — primary objective)
β = 0.3 (imbalance — secondary objective)

makespan = maxv∈V( Σ execTime(t,v) )
imbalance = maxv(load) − minv(load)
